# Cadencia - GitHub Pages

Sitio estático limpio para GitHub Pages.

## Archivos necesarios

- `index.html`
- `assets/images/`
- `assets/gifs/`
- `.github/workflows/deploy-pages.yml`
- `.nojekyll`

## Cómo publicarlo

1. Descomprime este ZIP.
2. En GitHub, sube TODO lo que está dentro de la carpeta descomprimida a la raíz del repositorio.
3. Ve a `Settings → Pages`.
4. En `Build and deployment`, selecciona `GitHub Actions`.
5. Ve a `Actions` y espera a que termine `Deploy Cadencia to GitHub Pages`.

No subas el ZIP completo al repo. Deben quedar visibles `index.html` y la carpeta `assets` en la raíz.
