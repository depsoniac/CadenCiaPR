Formato de proyecto

.msub es el corazón de Cadencia.

Un SRT solo guarda texto y tiempos. Un proyecto .msub conserva el trabajo completo: pistas, etiquetas, marcadores, estado de edición, capturas, referencias al material y organización del timeline.

Proyecto vivo: cierra y vuelve después sin perder estructura ni contexto.
Multipista real: guarda hablantes separados, orden y capas de trabajo.
Más que subtítulos: conserva marcadores, notas, capturas y ajustes del flujo.
Salida compatible: edita en .msub y exporta SRT cuando necesites entregar.
Inicio flexible

Importa video, abre .msub o combina varias pistas SRT.

El formato .msub funciona como contenedor de proyecto: guarda el estado completo para que el editor no tenga que reconstruir pistas, etiquetas y marcadores cada vez.

Timeline multipista

No es una lista de subtítulos. Es una línea de tiempo.

Visualiza pistas, waveform, playhead, zoom y bloques de texto como si editaras clips. Perfecto para materiales largos con varios hablantes y muchas correcciones.

Movimiento preciso

Mueve múltiples pistas como clips.

Selecciona bloques grandes, desplázalos y conserva el orden visual para ajustar diálogos enteros sin hacerlo subtítulo por subtítulo.

Etiquetas vivas

Las etiquetas cambian con la pista.

Al mover un subtítulo de pista, Cadencia actualiza el hablante en tiempo real. Menos edición manual, menos errores repetidos.

Edición rápida con memoria muscular de editor.

Borra, corta, une, reemplaza, copia, pega y deshaz usando gestos y atajos que se sienten naturales si vienes de edición de video.

⌫

Borrar y deshacer sin miedo

Elimina subtítulos rápido y usa combinaciones conocidas como Ctrl+Z, Ctrl+C y Ctrl+V para corregir sin romper el ritmo.

✂

Corte preciso por palabra

El cursor de texto sigue al playhead para saber exactamente en qué palabra estás al dividir un subtítulo.

↔

Unir frases sin ensuciar etiquetas

Combina subtítulos contiguos y deja una sola etiqueta limpia, como si siempre hubieran sido uno.

Atajos conocidos

Borra, copia, pega y deshaz.

Operaciones rápidas para trabajar cientos de subtítulos sin pelearte con la interfaz.

Corte fino

Divide textos en el punto exacto.

Unión automática

Une dos subs y conserva una sola etiqueta.

Ideal para limpiar cortes innecesarios y mantener una lectura natural.

Fusión rápida

Automatiza microcorrecciones repetitivas.

Acciones pequeñas como unir, ajustar y limpiar texto se resuelven en segundos; repetidas cientos de veces, se convierten en horas recuperadas.

Automatizaciones para casos reales

Menos trabajo manual, más control visual.

Cadencia no solo muestra subtítulos: entiende pistas, etiquetas, solapamientos, reemplazos frecuentes y texto sensible para ayudarte a limpiar proyectos grandes.

Anti-solapamiento: evita que dos textos se encimen en pantalla.
Etiquetas automáticas: el hablante se actualiza al mover de pista.
Textos rápidos: reemplaza selecciones desde una lista editable.
Censuras detectadas: ubica contenido sensible marcado con asteriscos o etiquetas.
Sistema anti solapamientos

El texto nunca se tapa encima de otro.

Si dos subtítulos coinciden, Cadencia acomoda el más pequeño en otro renglón para conservar lectura limpia.

Texto rápido

Reemplaza texto seleccionado en segundos.

Usa una lista editable para insertar frases recurrentes como etiquetas, redacciones, inaudibles o notas de privacidad.

Revisión sensible

Detecta censuras y marcas manuales.

Encuentra groserías censuradas, asteriscos y etiquetas como [Redacted for Privacy] para revisar material delicado con menos riesgo.

Transcripción local

Subtítulos base con detección de hablantes.

Cadencia puede generar una primera versión de subtítulos y separar voces en pistas para acelerar el armado inicial. En audios claros puede alcanzar hasta un 80% de precisión como punto de partida, dejando la revisión fina en manos del editor.

Ideal para equipos que guardan proyectos y material pesado fuera del escritorio local.
